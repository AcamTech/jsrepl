#jsREPL

A client-side REPL engine for languages implemented or compiled to JavaScript.

##Current Languages

* JavaScript Variants
  * JavaScript
  * CoffeeScript
  * Kaffeine
  * Move
  * Traceur (JavaScript.next)

* Esoteric
  * Bloop
  * Brainfuck
  * LOLCODE
  * Unlambda
  * Emoticon

* Classic
  * Quick Basic
  * Forth
  * Smalltalk (GUI-less/Incomplete)

* Serious
  * Scheme
  * Lua
  * Lisp (to be replaced)
  * Python
  * Ruby
