#jsREPL

A one-page client-side REPL for many languages.

##Current Languages

* JavaScript Variants
  * JavaScript
  * CoffeeScript
  * Kaffeine
  * Move
  * Traceur (JavaScript.next)

* Esoteric
  * Bloop
  * Brainfuck
  * LOLCODE
  * Unlambda
  * Emoticon

* Classic
  * Quick Basic
  * Forth
  * Smalltalk (GUI-less)

* Serious
  * Scheme
  * Lua
  * Lisp (to be replace)
