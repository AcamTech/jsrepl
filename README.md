#jsREPL
##In a nutshell
A collection of language interpreters written in JavaScript to be put on a single website to give an easy to use one page REPL to all of them.

##Languages
<ul>
  <li>Quick Basic</li>
  <li>Scheme</li>
  <li>Lisp</li>
  <li>CoffeeScript</li>
  <li>Ruby</li>
  <li>BrainFuck</li>
</ul>
